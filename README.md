# 🧠 gAIde AI 모델 서버
> gAIde에서 백엔드의 요청을 받아 AI 모델에 요청하고, 받아낸 응답을 다시 백엔드로 전달해주는 역할을 합니다.

## 파일 구조
<pre>
<code>
  src/
  ├── prompters/
  │   ├── __init__.py
  │   ├── evaluation.py
  │   ├── feedback.py
  │   └── outline.py
  ├── enums.py
  ├── routes.py
  ├── schemas.py
  main.py
  settings.py
</code>
</pre>

## 프롬프터 설명
- feedback.py
  - 사용자의 글을 입력받아 피드백하는 문장을 응답해줍니다. <br>
    ex) '저는 어렸을 때부터 컴퓨터를 좋아했고...' - 이 부분에 대해 자세한 경험이나 설명이 부족해요. 개인의 경험과 엮을 수 있다면 더 좋은 자기소개서가 될 것 같아요.
- outline.py
  - 사용자의 글을 입력받아 전체적인 개요를 작성하여 응답해줍니다. <br>
    ex) 사용자의 글에 대하여, 서론 - '자신의 경험 소개', 본론 - '자신의 경력 소개', 결론 - '자신의 다짐 소개'
- evaluation.py
  - 사용자가 글을 완성하여 저장 버튼을 눌렀을 때, 사용자가 작성한 글을 바탕으로 최종 평가를 해줍니다. <br>
    ex) 이 글은 자신의 장점을 소개하는 데에는 효과적이지만, 자신이 어떤 일을 맡고 싶은지에 대해서는 설명이 부족해요.
